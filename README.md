# KinoVzor - Мобильная кинотека с рецензиями

Приложение для осмотра и рецензирования фильмов.

## Основные функции

- Авторизация и регистрация пользователей
- Опускание фильмов с рейтингом
- Написание рецензий
- Оставление оценок (звездочки)
- Добавление фильмов в избранное
- Фильтрация по жанрам
- Сортировка по рейтингу, югоду, названию
- Панель модератора
- Административная панель

## Технологии

- **Backend**: FastAPI
- **Database**: SQLite + SQLAlchemy ORM (async)
- **Database Migrations**: Alembic
- **Frontend**: Vanilla JavaScript
- **Styling**: CSS

## Установка

### Предположения

- Python 3.8+
- pip

### Шаги по установке

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Paketiki/final.git
cd final
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate    # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Нарботание миграций (создаст базу SQLite автоматически):
```bash
cd app
alembic upgrade head
cd ..
```

5. Запустите приложение:
```bash
python app/main.py
```

Приложение будет доступно по адресу `http://localhost:8000`

## Как пользовать

1. При первом запуске зарегистрируйтесь или войдите как гость
2. Осматривайте фильмы, фильтруйте по жанрам
3. Нажимайте на постер для воскрытия деталей фильма
4. Оставляйте оценки и рецензии (авторизанные пользователи)
5. Добавляйте фильмы в избранное

## Роли пользователей

- **Простые пользователи**: чтение, оставление оценок и рецензий
- **Модераторы**: одобрение/удаление рецензий
- **Админы**: экспорт и добавление фильмов
